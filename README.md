# Wanderlust-Airbnb-Clone

A learning project replicating core Airbnb features with full CRUD functionality. Built using **Node.js, Express, MongoDB, and EJS templates**.

## Features

- View all property listings (Read)
- Add new listings (Create)
- View single listing details (Read)
- Edit listings (Update)
- Delete listings (Delete)
- User-friendly UI using EJS templates
- Method override to handle PUT and DELETE requests
- Organized routes and views

## Tech Stack

- Node.js
- Express.js
- MongoDB with Mongoose
- EJS templates
- Method-override
- HTML/CSS for frontend
